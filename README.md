## Frontend Stack

- Angular11
- NGRX State Management
- RxJS
- NGX-Rocket starter kit
- NG-Bootstrap
- GitHub (CI/CD pipeline)
- deploy to FTP (dev, master)
- Cypress integration testing

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ npm run start
```

http://localhost:422000

## Testing: cypress

```bash
$ npm run cypress:open
```
